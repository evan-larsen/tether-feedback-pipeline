# Taxonomy

Use one of the following labels:

## `feature_request`

- `audio`
- `widget`
- `notifications`
- `social`
- `leaderboard`
- `favorites`
- `content_scope`
- `content_navigation`
- `integration`
- `streaks`
- `blocking_tools`
- `other`

Use when the user is requesting a capability, enhancement, or additional content.

Examples:
- Wants audio reading of verses: `feature_request|audio`
- Wants a home-screen streak widget: `feature_request|widget`
- Wants Bible or other standard works added: `feature_request|content_scope`

## `bug_report`

- `scrolling`
- `install`
- `access`
- `content_loading`
- `linking`
- `other`

Use when the user says something is not working as expected.

Examples:
- App will not scroll/read on phone: `bug_report|scrolling`
- Can’t open or find the app after being told it exists: `bug_report|access`

## `confusion`

- `availability`
- `how_to_use`
- `content_scope`
- `where_is_link`
- `other`

Use when the user seems confused about access, availability, or usage, but not clearly reporting a defect.

Examples:
- Asking whether Android is available yet: `confusion|availability`
- Asking why only Nephi is selectable when they expected more: `confusion|content_scope`

## `praise`

- `general`
- `value`

Use when the message is mainly positive sentiment.

Examples:
- “This is so cool”: `praise|general`
- “This will help me stay consistent”: `praise|value`

## `tester_interest`

- `android`
- `beta_access`
- `app_link`
- `general`

Use when the main point is wanting access, wanting to test, or asking for the app link.

Examples:
- “I’d love to test it on Android”: `tester_interest|android`
- “Can you send me the app link?”: `tester_interest|app_link`

## `growth_signal`

- `influencer`
- `promotion`
- `partnership`
- `referral`
- `other`

Use when the user offers growth help, distribution, or network leverage.

Examples:
- Says they have influencer connections: `growth_signal|influencer`

## `non_product`

- `social`
- `joke`
- `spam`
- `other`

Use when the line does not provide meaningful product feedback or product-relevant intent.

Examples:
- Pure joke replies: `non_product|joke`

## `other`

- `unclear`
- `mixed`

Use when the line is product-relevant but does not fit confidently elsewhere.

Examples:
- Mixed unclear sentiment without a clear ask: `other|mixed`
